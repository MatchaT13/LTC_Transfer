# LTC Transfer System

Spring Boot prototype for hospital-to-LTC transfer management.

## Requirements
- Java 17
- Maven

## How to Run

1. Open the project folder in VS Code or IntelliJ.
2. Open a terminal in the project root (the folder with `pom.xml`).
3. Run:

```bash
mvn install
mvn spring-boot:run
  When done open up in you local browse: http://localhost:8080/login
